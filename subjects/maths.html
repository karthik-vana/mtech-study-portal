Q1(a) NUMERICAL Find E(X) for f(x) = (1/2)x, 0<x<2; 0 otherwise 8M

🔹 Given

Probability density function (PDF):

f(x) = (1/2)x, 0 < x < 2 f(x) = 0, otherwise

Find: E(X) — the expected value (mean) of X

🔹 Formula Used

E(X) = ∫₋∞^∞ x · f(x) dx

Since f(x) is non-zero only for 0 < x < 2:

E(X) = ∫₀² x · f(x) dx

🔹 Step-by-Step Solution

Step 1: Verify it's a valid PDF (normalization check):

∫₀² (1/2)x dx = (1/2) · [x²/2]₀² = (1/2) · (4/2 − 0) = (1/2) · 2 = 1 ✓ Valid PDF

Step 2: Substitute f(x) = (1/2)x into E(X) formula:

E(X) = ∫₀² x · (1/2)x dx = (1/2) ∫₀² x² dx

Step 3: Evaluate the integral using power rule ∫xⁿdx = xⁿ⁺¹/(n+1):

E(X) = (1/2) · [x³/3]₀² = (1/2) · (2³/3 − 0³/3) = (1/2) · (8/3) = 8/6 = 4/3

🔹 Final Answer

E(X) = 4/3 ≈ 1.333

The average value of X with this triangular distribution is 4/3.

🧠 Concept Insight

E(X) is the "center of mass" of the probability distribution. For a continuous RV, it's computed via integration of x·f(x). Since f(x) increases linearly, the distribution is right-skewed, pushing the mean above the midpoint (1).

💡 Memory Tip

"E(X) for continuous = ∫ x·f(x) dx" — think: weighted average where weight = probability density.

Q1(b) THEORY Define Probability Mass Function and Density Function 7M

📌 Probability Mass Function (PMF)

  • Definition: A function p(x) that gives the probability that a discrete random variable X takes the exact value x.
  • Notation: p(x) = P(X = x)
  • Properties:
    • p(x) ≥ 0 for all x
    • Σ p(xᵢ) = 1 (sum over all possible values)
    • P(X ∈ A) = Σₓ∈ₐ p(x)
  • Example: Rolling a fair die: p(x) = 1/6 for x = 1,2,3,4,5,6
PMF Example (Coin Toss, X = No. of Heads in 2 tosses): P(X=0) = 1/4, P(X=1) = 2/4, P(X=2) = 1/4 Sum = 1/4 + 2/4 + 1/4 = 1 ✓

📌 Probability Density Function (PDF)

  • Definition: A function f(x) for a continuous random variable X such that probabilities over intervals are computed by integration.
  • Key Formula: P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx
  • Properties:
    • f(x) ≥ 0 for all x
    • ∫₋∞^∞ f(x) dx = 1
    • P(X = c) = 0 for any specific value c (continuous!)
  • Note: f(x) is NOT a probability itself — it's a density. It can be > 1.

Q3(b) NUMERICAL Define Skewness & Kurtosis; Karl Pearson's coefficient for 25,15,23,40,27,25,23,25,20 8M

📌 Part (i) — Skewness & Kurtosis Definitions

  • Skewness: Measures the asymmetry of a distribution about its mean.
    • Positive skew: Tail extends to the right (Mean > Median > Mode)
    • Negative skew: Tail extends to the left (Mean < Median < Mode)
    • Zero skew: Symmetric distribution
  • Kurtosis: Measures the "peakedness" or tailedness of distribution.
    • Leptokurtic: β₂ > 3 (more peaked than Normal)
    • Mesokurtic: β₂ = 3 (Normal distribution)
    • Platykurtic: β₂ < 3 (flatter than Normal)
Karl Pearson's Coefficient of Skewness: Sk = (Mean − Mode) / SD Kurtosis: β₂ = μ₄ / μ₂² (μ₂, μ₄ are central moments)

🔹 Part (ii) — Given Data

Data: 25, 15, 23, 40, 27, 25, 23, 25, 20  |  n = 9

🔹 Step-by-Step Calculation

Step 1: Sort data: 15, 20, 23, 23, 25, 25, 25, 27, 40

Step 2: Find Mode (most frequent value):

25 appears 3 times → Mode = 25

Step 3: Calculate Mean:

Mean = (25+15+23+40+27+25+23+25+20) / 9 = 223 / 9 = 24.78

Step 4: Calculate Standard Deviation:

Deviations from mean (x − x̄): x: 25 15 23 40 27 25 23 25 20 x−x̄: 0.22 -9.78 -1.78 15.22 2.22 0.22 -1.78 0.22 -4.78 (x−x̄)²: 0.048 95.65 3.168 231.65 4.928 0.048 3.168 0.048 22.85 Σ(x−x̄)² = 0.048+95.65+3.168+231.65+4.928+0.048+3.168+0.048+22.85 = 361.56 Variance σ² = Σ(x−x̄)² / n = 361.56 / 9 = 40.17 SD σ = √40.17 = 6.338

Step 5: Karl Pearson's Coefficient of Skewness:

Sk = (Mean − Mode) / SD = (24.78 − 25) / 6.338 = −0.22 / 6.338 = −0.0347

🔹 Final Answers

Mean = 24.78, Mode = 25, SD = 6.338

Karl Pearson's Skewness = −0.035 ≈ 0

Since Sk ≈ 0, the distribution is nearly symmetric (very slight negative skew).

Q4(a) NUMERICAL PDF f(x) = kxe^(-λx) for x≥0 — Find K, Mean, Variance 8M

🔹 Given

f(x) = k·x·e^(−λx), x ≥ 0, λ > 0 f(x) = 0, otherwise

Find: (i) K   (ii) Mean   (iii) Variance

🔹 Key Formula — Gamma Integral

∫₀^∞ xⁿ e^(−λx) dx = n! / λⁿ⁺¹ (for n ≥ 0, λ > 0) Special cases: n=0: ∫₀^∞ e^(−λx) dx = 1/λ n=1: ∫₀^∞ x·e^(−λx) dx = 1/λ² n=2: ∫₀^∞ x²·e^(−λx) dx = 2/λ³ n=3: ∫₀^∞ x³·e^(−λx) dx = 6/λ⁴

🔹 Part (i): Find K using Normalization Condition

∫₀^∞ f(x) dx = 1 ∫₀^∞ k·x·e^(−λx) dx = 1 k · ∫₀^∞ x·e^(−λx) dx = 1 k · (1/λ²) = 1 [using gamma integral with n=1] ∴ k = λ²

K = λ²

So f(x) = λ²·x·e^(−λx) — this is the Gamma distribution Γ(2, λ)

🔹 Part (ii): Mean E(X)

E(X) = ∫₀^∞ x · f(x) dx = ∫₀^∞ x · λ²·x·e^(−λx) dx = λ² ∫₀^∞ x²·e^(−λx) dx = λ² · (2/λ³) [gamma integral, n=2] = 2λ²/λ³ = 2/λ

Mean E(X) = 2/λ

🔹 Part (iii): Variance Var(X)

Step 1: Find E(X²):

E(X²) = ∫₀^∞ x² · λ²·x·e^(−λx) dx = λ² ∫₀^∞ x³·e^(−λx) dx = λ² · (6/λ⁴) [gamma integral, n=3] = 6λ²/λ⁴ = 6/λ²

Step 2: Calculate Variance:

Var(X) = E(X²) − [E(X)]² = 6/λ² − (2/λ)² = 6/λ² − 4/λ² = 2/λ²

Variance Var(X) = 2/λ²

🔹 Final Answers

(i) K = λ²

(ii) Mean E(X) = 2/λ

(iii) Variance Var(X) = 2/λ²

This is a Gamma(2, λ) distribution — also known as Erlang-2 or chi-squared(4)/2.

🧠 Concept Insight

The Gamma integral ∫₀^∞ xⁿe^(−λx)dx = n!/λⁿ⁺¹ is the KEY tool for all exponential-family integrals. Memorize n=1,2,3 cases. Standard deviation σ = √2/λ.

Q4(b) THEORY Define Central Limit Theorem with Example 7M

📌 Central Limit Theorem (CLT)

  • Statement: If X₁, X₂, …, Xₙ are independent and identically distributed (i.i.d.) random variables with mean μ and finite variance σ², then as n → ∞, the sample mean X̄ follows a Normal distribution:
X̄ ~ N(μ, σ²/n) as n → ∞ Standardized form: Z = (X̄ − μ) / (σ/√n) ~ N(0,1)
  • Works regardless of the shape of the original distribution
  • Practically reliable when n ≥ 30
  • Foundation of all hypothesis testing and confidence intervals

📌 Example

Scores in an exam follow a distribution with μ = 60, σ = 10. Find P(X̄ > 62) for a sample of n = 25.

Standard Error = σ/√n = 10/√25 = 10/5 = 2 Z = (X̄ − μ) / (σ/√n) = (62 − 60) / 2 = 1.0 P(X̄ > 62) = P(Z > 1.0) = 1 − Φ(1.0) = 1 − 0.8413 = 0.1587 ≈ 15.87%

💡 Memory Tip

"No matter the population shape — with large enough sample, the sample mean is NORMAL." The larger n, the better the normal approximation.

Q5(a) THEORY Write briefly about Markov Chains Process 7M

📌 Markov Chain — Definition

  • A stochastic process {Xₙ, n = 0,1,2,…} taking values in a countable state space S = {s₁, s₂, …}
  • Markov Property (Memoryless): The future state depends ONLY on the present state, not the past history.
P(Xₙ₊₁ = j | X₀=i₀, X₁=i₁, …, Xₙ=i) = P(Xₙ₊₁=j | Xₙ=i) = pᵢⱼ (transition probability)

📌 Transition Matrix

  • P = [pᵢⱼ] is the transition probability matrix where pᵢⱼ = P(going from state i to state j)
  • Each row sums to 1: Σⱼ pᵢⱼ = 1 (stochastic matrix)
  • All entries pᵢⱼ ≥ 0
Example: 2-state chain S = {1, 2} State 1 State 2 P = [ 0.7 0.3 ] ← from State 1 [ 0.4 0.6 ] ← from State 2 Row sums: 0.7+0.3=1, 0.4+0.6=1 ✓

📌 Key Properties

  • Irreducible: Can reach any state from any state
  • Aperiodic: No fixed cycles between states
  • Stationary Distribution π: πP = π (long-run behavior)
  • Ergodic Chain: Irreducible + Aperiodic → unique stationary distribution exists
  • n-step transition: P(n) = Pⁿ (matrix power)

💡 Memory Tip

"Markov chains have no memory — only the CURRENT state matters for the FUTURE." Like flipping coins: past results don't affect the next flip.

Q2(b)NUMERICAL Moments about value 4: −1.5, 17, −30, 108 → moments about mean, β₁, β₂, origin 8M

🔹 Given

Moments about A = 4: μ₁' = −1.5, μ₂' = 17, μ₃' = −30, μ₄' = 108

Let d = μ₁' (first moment about A) μ₁ = 0 (always) μ₂ = μ₂' − (μ₁')² μ₃ = μ₃' − 3μ₂'·μ₁' + 2(μ₁')³ μ₄ = μ₄' − 4μ₃'·μ₁' + 6μ₂'·(μ₁')² − 3(μ₁')⁴

Step 1: μ₁' = −1.5, so mean = A + μ₁' = 4 + (−1.5) = 2.5

Step 2: μ₂ (Variance about mean):

μ₂ = μ₂' − (μ₁')² = 17 − (−1.5)² = 17 − 2.25 = 14.75

Step 3: μ₃:

μ₃ = μ₃' − 3μ₂'·μ₁' + 2(μ₁')³ = −30 − 3(17)(−1.5) + 2(−1.5)³ = −30 + 76.5 + 2(−3.375) = −30 + 76.5 − 6.75 = 39.75

Step 4: μ₄:

μ₄ = μ₄' − 4μ₃'·μ₁' + 6μ₂'·(μ₁')² − 3(μ₁')⁴ = 108 − 4(−30)(−1.5) + 6(17)(−1.5)² − 3(−1.5)⁴ = 108 − 180 + 6(17)(2.25) − 3(5.0625) = 108 − 180 + 229.5 − 15.1875 = 142.3125

Q3(b)THEORY Show that sample mean x̄ is an unbiased estimate for population mean μ 8M

📌 Definition: Unbiased Estimator

An estimator T is unbiased for parameter θ if:

E(T) = θ for all possible values of θ

📌 Proof: E(x̄) = μ

Step 1: Let X₁, X₂, …, Xₙ be a random sample from a population with mean μ. The sample mean is:

x̄ = (X₁ + X₂ + … + Xₙ) / n = (1/n) Σᵢ₌₁ⁿ Xᵢ

Step 2: Take expectation of x̄:

Q5(b)THEORY Explain Maximum Likelihood Estimation (MLE) and its properties 8M

📌 Maximum Likelihood Estimation (MLE)

  • Principle: Find the parameter value θ that maximizes the probability (likelihood) of observing the given sample data.
  • Likelihood Function: L(θ) = P(X₁=x₁, X₂=x₂, …, Xₙ=xₙ | θ)
  • For independent samples: L(θ) = Π f(xᵢ; θ)
  • For computational convenience, we maximize the Log-Likelihood: ℓ(θ) = ln L(θ)
MLE Steps: 1. Write likelihood L(θ) = Πᵢ f(xᵢ; θ) 2. Take log: ℓ(θ) = Σᵢ ln f(xᵢ; θ) 3. Differentiate: dℓ/dθ = 0 (Score equation) 4. Solve for θ̂ (MLE estimate) 5. Verify: d²ℓ/dθ² < 0 (maximum, not minimum)

📌 Example: MLE for Normal Distribution

📌 Statistical Hypothesis

  • A statistical hypothesis is a statement or assumption about a population parameter (μ, σ, p) that can be tested using sample data.
  • Null Hypothesis (H₀): The statement of "no effect" or "no difference" — assumed true until evidence says otherwise. Example: H₀: μ = 50

📌 Steps in Testing a Hypothesis

  • A square matrix A of order n×n (n = number of vertices)
  • A[i][j] = 1 if there is an edge between vertex i and vertex j; 0 otherwise

Q1(b)THEORY Define Eulerian circuit and Hamiltonian circuit; example of graph with neither 8M

📌 Eulerian Circuit

  • Definition: A closed walk that visits every edge exactly once and returns to the starting vertex.
  • Condition (Euler's Theorem): A connected graph has an Eulerian circuit iff every vertex has even degree.
  • Eulerian Path (not circuit): Exists iff exactly 2 vertices have odd degree.
  • Example: Königsberg Bridge Problem — 4 vertices all with odd degree → no Eulerian circuit.

📌 Hamiltonian Circuit

  • Definition: A closed path that visits every vertex exactly once and returns to the starting vertex.
  • No simple necessary & sufficient condition (NP-complete problem)
  • Ore's Theorem: If deg(u) + deg(v) ≥ n for all non-adjacent pairs, Hamiltonian circuit exists.
  • Example: K_n (complete graph) always has a Hamiltonian circuit.

📌 Graph with NEITHER Eulerian nor Hamiltonian Circuit

Consider a Path Graph P₄: 1 — 2 — 3 — 4 Degrees: deg(1)=1 (odd), deg(2)=2, deg(3)=2, deg(4)=1 (odd) Eulerian Circuit? NO — vertices 1 and 4 have odd degree (Need all even degrees for Eulerian circuit) Hamiltonian Circuit? NO — path graph is not cyclic; a Hamiltonian circuit would need to return to start, but P₄ has no cycle. ∴ P₄ has neither an Eulerian circuit nor a Hamiltonian circuit.

Q2(b)THEORY Explain Graph Coloring and Chromatic Number with example 8M

📌 Graph Coloring

  • Assignment of colors to vertices such that no two adjacent vertices share the same color.
  • Formally: A k-coloring is a function f: V → {1,2,…,k} such that f(u) ≠ f(v) whenever (u,v) ∈ E.
  • A graph is k-colorable if such an assignment exists with k colors.

📌 Chromatic Number χ(G)

χ(G) = minimum number of colors needed for proper coloring Key Results: • Bipartite graph: χ = 2 • Tree: χ = 2 • Cycle with even vertices: χ = 2 • Cycle with odd vertices: χ = 3 • Complete graph Kₙ: χ = n • Planar graph (4-Color Theorem): χ ≤ 4

📌 Example: Cycle Graph C₅ (Pentagon)

Vertices: 1-2-3-4-5-1 Try 2 colors: 1=R, 2=B, 3=R, 4=B, 5=R But 5 and 1 are adjacent, both R → CONFLICT! Try 3 colors: 1=R, 2=B, 3=R, 4=B, 5=G Check all edges: 1-2(R,B)✓, 2-3(B,R)✓, 3-4(R,B)✓, 4-5(B,G)✓, 5-1(G,R)✓ All OK! ∴ χ(C₅) = 3

Q3(b)THEORY Define Minimum Spanning Tree; describe Prim's Algorithm with example 8M

📌 Minimum Spanning Tree (MST)

  • A spanning tree includes all vertices with exactly (V−1) edges and no cycles.
  • A Minimum Spanning Tree is the spanning tree with minimum total edge weight.
  • Properties: V vertices, V−1 edges, connected, acyclic

📌 Prim's Algorithm

Input: Weighted connected graph G Output: MST T Algorithm: 1. Start with any vertex s, add to MST set 2. REPEAT until all vertices included: a. Find the minimum weight edge connecting a vertex IN MST to a vertex NOT IN MST b. Add that edge and vertex to MST 3. Output MST Time Complexity: O(V²) basic | O(E log V) with heap

📌 Example

Q4(a)THEORY Show that complete bipartite graph K₃,₃ is not planar 7M

📌 Key Theorem for Bipartite Planar Graphs

For a simple connected BIPARTITE planar graph: E ≤ 2V − 4 (Because bipartite graphs have no odd cycles → minimum cycle length is 4, so each face has ≥ 4 edges)

🔹 Proof K₃,₃ is Non-Planar

For K₃,₃: V = 6 (3 in each partition) E = 3 × 3 = 9 (complete bipartite) Apply the bipartite planarity condition: E ≤ 2V − 4 9 ≤ 2(6) − 4 9 ≤ 8 CONTRADICTION! 9 > 8 ∴ K₃,₃ VIOLATES the necessary condition for planarity. ∴ K₃,₃ is NOT PLANAR. QED ✓

🧠 Physical Interpretation

K₃,₃ is the "water-gas-electricity" problem: connect 3 houses to 3 utilities (gas, water, electric) without crossing pipes/wires — impossible in a plane!

Q4(b)THEORY Prove that the complete graph K₅ is non-planar 8M

📌 Euler's Planarity Condition

For a simple connected planar graph: E ≤ 3V − 6 (Every face has ≥ 3 edges, and by Euler's formula V-E+F=2)

🔹 Proof K₅ is Non-Planar

For K₅ (complete graph on 5 vertices): V = 5 E = C(5,2) = 5×4/2 = 10 Apply planarity condition: E ≤ 3V − 6 10 ≤ 3(5) − 6 10 ≤ 15 − 6 10 ≤ 9 CONTRADICTION! 10 > 9 ∴ K₅ violates the necessary condition for planarity. ∴ K₅ is NOT PLANAR. A graph is planar iff it contains no subdivision of K₅ or K₃,₃ as a subgraph. K₅ and K₃,₃ are the two fundamental non-planar graphs.
